Functions

A function is a collection of pairs of numbers with the following property: if (a,b)(a,b) and (a,c)(a,c) are both in the collection, then b=cb = c; in other words, the collection must not contain two different pairs with the same first element.

If ff is a function, the domain of ff is the set of all aa for which there is some bb such that (a,b)(a,b) is in ff. If aa is in the domain of ff, it follows from definition of a function that there is, in fact, a unique number bb such that (a,b)(a,b) is in ff. This unique bb is denoted by f(a)f(a).
