Definition

For sets AA and BB, a function ff from AA to BB is a rule that assigns to each x∈Ax \in A a unique value f(x)∈Bf(x) \in B. More precisely, a function from AA to BB is a set f⊆A×Bf \subseteq A \times B such that for every x∈Ax \in A, there is a unique y∈By \in B with (x,y)∈f(x, y) \in f. Of course therefore, if (x,y)∈f(x, y) \in f then we can write f(x)=yf(x) = y. Here are some examples.

  1. f:ℝ→ℝf\colon \mathbb R \to \mathbb R given by f(x)=x2f(x) = x^2, or using an alternative notation, x↦x2x \mapsto x^2 is a function.

  2. A non-example is f:ℝ→ℝf\colon \mathbb R \to \mathbb R given by f(x)=1xf(x) = \frac{1}{x} since it is undefined at x=0x=0.

  3. Another non-example is f:ℝ→ℝf\colon \mathbb R \to \mathbb R given by $f(x) = \pm \sqrt{\abs{x}}$ since it does not define a unique value in the output space for a given input, such as x=2x=2.

  4. f:ℝ→ℝf\colon \mathbb R \to \mathbb R given by f(x)={1x∈ℚ0otherwisef(x) = \begin{cases} 1 & x \in \mathbb Q \\ 0 & \text{otherwise} \end{cases} is a function since it clearly satisfies the second definition. Note that even though we don’t know if e+πe + \pi is rational or not, the function is still well defined since it produces a unique solution for f(e+π)f(e + \pi), we just don’t know which output value it gives.

  5. A={1,2,3,4,5}A = \{ 1, 2, 3, 4, 5 \}, B={1,2,3,4}B = \{ 1, 2, 3, 4 \}, and f:A→Bf\colon A \to B is given by f(1)=1f(2)=4f(3)=3f(4)=3f(5)=4\begin{align*} f(1) & = 1 \\ f(2) & = 4 \\ f(3) & = 3 \\ f(4) & = 3 \\ f(5) & = 4 \end{align*}

  6. A={1,2,3}A = \{ 1, 2, 3 \}, f:A→Af\colon A \to A is given by f(1)=1f(2)=3f(3)=2\begin{align*} f(1) & = 1 \\ f(2) & = 3 \\ f(3) & = 2 \end{align*}

  7. A={1,2,3,4}A = \{ 1, 2, 3, 4 \}, f:A→Af\colon A \to A is given by f(1)=1f(2)=3f(3)=3f(4)=4\begin{align*} f(1) & = 1 \\ f(2) & = 3 \\ f(3) & = 3 \\ f(4) & = 4 \end{align*}

  8. A={1,2,3,4}A = \{ 1, 2, 3, 4 \}, B={1,2,3}B = \{ 1, 2, 3 \}, f:A→Bf\colon A \to B is given by f(1)=3f(2)=3f(3)=2f(4)=1\begin{align*} f(1) & = 3 \\ f(2) & = 3 \\ f(3) & = 2 \\ f(4) & = 1 \end{align*}

Injection, surjection and bijection

Definition

A function f:A→Bf\colon A \to B is

  • injective, if ∀a,a′∈A\forall a, a' \in A, we have a≠a′⟹f(a)≠f(a′)a \neq a' \implies f(a) \neq f(a'), or equivalently, f(a)=f(a′)⟹a=a′f(a) = f(a') \implies a = a', or in words, ‘different points stay different’ (e.g. example 6 above).

  • surjective, if ∀b∈B\forall b \in B, ∃a∈A\exists a \in A such that f(a)=bf(a) = b, or in words, ‘everything in BB is hit’ (e.g. examples 6 and 8).

  • bijective, if it is injective and surjective, or in words, ‘everything in BB is hit exactly once’, or ‘ff pairs up elements of AA and elements of BB’ (e.g. example 6, or f:ℝ→ℝf\colon \mathbb R \to \mathbb R given by f(x)=x3f(x) = x^3).

Definition

For a function f:A→Bf\colon A \to B, AA is the domain, BB is the range, and $\{ b \in B : \exists a \in A \st f(a) = b \}$ is the image.

We must always provide the domain and range of a function; a function’s properties depend on this. For example, is the function ff defined by f(x)=f2f(x) = f^2 injective? If f:ℕ→ℕf\colon \mathbb N \to \mathbb N, then it is injective, but if f:ℤ→ℤf\colon \mathbb Z \to \mathbb Z, then it is not.

There are a number of properties that hold specifically for finite sets AA, BB:

  1. There is no surjection A→BA \to B if $\abs{B} > \abs{A}$.

  2. There is no injection A→BA \to B if $\abs{A} > \abs{B}$.

  3. For a function f:A→Af\colon A \to A, ff injective ⇔\iff ff surjective. Hence, if ff is either injective or surjective, it is bijective.

  4. There is no bijection from AA to any proper subset of AA.

As counterexamples for infinite sets:

  1. We define f0:ℕ→ℕf_0\colon \mathbb N \to \mathbb N by f0(x)=x+1f_0(x) = x+1. Then, f0f_0 is injective but not surjective.

  2. We define f1:ℕ→ℕf_1\colon \mathbb N \to \mathbb N by f0(x)=x−1f_0(x) = x-1, or 1 if x=1x=1. Then, f0f_0 is surjective but not injective.

  3. We define g:ℕ→ℕ\{1}g\colon \mathbb N \to \mathbb N \setminus \{ 1 \} by g(x)=x+1g(x) = x+1. Then, gg is bijective between ℕ\mathbb N and a proper subset of ℕ\mathbb N.

We provide some more examples of functions.

  1. For any set XX we have 1X:X→X1_X\colon X \to X defined by 1X(x)=x1_X(x) = x. This is known as the identity function on XX.

  2. For any set XX, and A⊂XA \subset X, we have an indicator function (or characteristic function) χA:X→{0,1}\chi_A\colon X \to \{ 0, 1 \} defined by χA(x)={0x∉A1x∈A\chi_A(x) = \begin{cases} 0 & x \notin A \\ 1 & x \in A \end{cases}

  3. A sequence of reals x1,x2,…x_1, x_2, \dots is a function f:ℕ→ℝf\colon \mathbb N \to \mathbb R defined by f(n)=xnf(n) = x_n.

  4. The operation ++ on ℕ\mathbb N is a function ℕ2→ℕ\mathbb N^2 \to \mathbb N.

  5. A set XX has size nn ⇔\iff there is a bijection between XX and {1,2,…,n}\{ 1, 2, \dots, n \}.

Composition of functions

Given f:A→Bf\colon A \to B and g:B→Cg\colon B \to C, we define the composition g∘f:A→Cg\circ f \colon A \to C, given by (g∘f)(a)=g(f(a))(g\circ f)(a) = g(f(a)). For example, if f:ℝ→ℝf\colon \mathbb R \to \mathbb R, f(x)=2xf(x) = 2x, g:ℝ→ℝg\colon \mathbb R \to \mathbb R, g(x)=x+1g(x) = x+1, then (f∘g)(x)=2(x+1)(f \circ g)(x) = 2(x+1), and (g∘f)(x)=2x+1(g \circ f)(x) = 2x + 1.

In general, the operation ∘\circ is not commutative, as we can see from this example. However, ∘\circ is associative. Given f:A→Bf\colon A \to B, g:B→Cg\colon B \to C, h:C→Dh\colon C \to D, we have h∘(g∘f)=(h∘g)∘fh \circ (g \circ f) = (h \circ g) \circ f. Indeed, for any input x∈Ax \in A, (h∘(g∘f))(x)=h((g∘f)(x))=h(g(f(x)))=(h∘g)(f(x))=((h∘g)∘f)(x)(h \circ (g \circ f))(x) = h((g \circ f)(x)) = h(g(f(x))) = (h \circ g)(f(x)) = ((h \circ g)\circ f)(x) Thus (h∘(g∘f))(x)=((h∘g)∘f)(x)(h \circ (g \circ f))(x) = ((h \circ g)\circ f)(x) for every x∈Ax \in A, so h∘(g∘f)=(h∘g)∘fh \circ (g \circ f) = (h \circ g)\circ f.

Invertibility

We say that a function f:A→Bf\colon A \to B is invertible if there exists some g:B→Ag\colon B \to A such that g∘f=1Ag \circ f = 1_A and f∘g=1Bf \circ g = 1_B. For example f:ℝ→ℝf\colon \mathbb R \to \mathbb R given by f(x)=2x+1f(x)=2x+1 has inverse g:ℝ→ℝg\colon \mathbb R \to \mathbb R given by g(x)=x−12g(x)=\frac{x-1}{2}. We can prove that this is correct by showing for all real numbers that (g∘f)(x)=x(g\circ f)(x) = x and vice versa as required.

As an example, consider f0:ℕ→ℕf_0\colon \mathbb N \to \mathbb N given by f0(x)=x+1f_0(x)=x+1, and f1:ℕ→ℕf_1\colon \mathbb N \to \mathbb N given by f1(x)=x−1f_1(x) = x-1 if x≠1x\neq 1 and 1 if x=1x=1. f1∘f0=1ℕf_1\circ f_0 = 1_{\mathbb N} but f0∘f1≠1ℕf_0\circ f_1 \neq 1_{\mathbb N} because they disagree at 1. So we must check inverses both ways.

In fact, f:A→Bf\colon A \to B is invertible if and only if it is a bijection.

  • (forward implication) Let gg be the inverse of ff. It is surjective because ∀b∈B\forall b \in B, we have b=f(g(b))b=f(g(b)). It is injective because given two elements a,a′a,a' such that f(a)=f(a′)f(a) = f(a'), we have g(f(a))=g(f(a′))=a=a′g(f(a)) = g(f(a')) = a = a' as required. So it is bijective.

  • (backward implication) Suppose ff is bijective. Let g(b)g(b) be the unique point a∈Aa \in A with f(a)=bf(a) = b for all b∈Bb \in B. Then this gg is the inverse of ff.

Relations

A relation on a set XX is a subset of R⊆X×XR \subseteq X \times X. We usually write aRbaRb to denote (a,b)∈R(a, b) \in R. Here are some examples.

  1. On ℕ\mathbb N, aRbaRb if a≡b(5)a \equiv b\ (5). For example, 2R122R12 but not 2R112R11.

  2. On ℕ\mathbb N, aRbaRb if a∣ba \mid b.

  3. On ℕ\mathbb N, aRbaRb if a≠ba \neq b.

  4. On ℕ\mathbb N, aRbaRb if a=b±1a=b \pm 1.

  5. On ℕ\mathbb N, aRbaRb if $\abs{a-b} \leq 2$.

  6. On ℕ\mathbb N, aRbaRb if either a,b≤6a, b \leq 6 or a,b>6a, b > 6.

A relation may have a number of important properties:

  • (reflexive) If ∀x∈X\forall x \in X, xRxxRx, e.g. examples 1, 2, 5, 6.

  • (symmetric) If ∀x,y∈X\forall x, y \in X, xRy⟹yRxxRy \implies yRx, e.g. examples 1, 3, 4, 5, 6.

  • (transitive) If ∀x,y,z∈X\forall x, y, z \in X, xRy,yRz⟹xRzxRy, yRz \implies xRz, e.g. examples 1, 2, 6.

An equivalence relation is a relation that is reflexive, symmetric and transitive. Examples 1, 6 above are equivalence relations. Here are some more examples.

  1. On ℕ\mathbb N, xRyxRy if x=yx=y.

  2. Considering a partition of set XX into subsets C1,C2,…,i∈IC_1, C_2, \dots, i \in I where the CiC_i are non-empty and disjoint, and their union is XX. Then consider the relation aRbaRb if ∃i\exists i such that a∈Cia \in C_i and b∈Cib \in C_i. aRbaRb is an equivalence relation. In fact, all equivalence relations can be considered to be in this form; we will prove this shortly.

For an equivalence relation RR on a set XX, and x∈Xx \in X, we define the equivalence class [x]={y∈X:yRx}[x] = \{ y \in X: y R x \}. In the first example 1 above, [2]={y∈ℕ:y≡2(5)}[2] = \{ y \in \mathbb N : y \equiv 2\ (5) \}.

Equivalence classes as partitions

Proposition

Let RR be an equivalence relation on a set XX. Then the equivalence classes of RR partition XX.

Proof

Each equivalence class [x][x] is non-empty, since x=xx = x. Further, ⋃x∈X=X\bigcup_{x \in X} = X since x∈[x]x \in [x] for all x∈Xx \in X. Now we must show that the classes are disjoint, or are equal. Given x,yx, y with [x]∩[y]≠⌀[x] \cap [y] \neq \varnothing, we need to show that [x]=[y][x] = [y]. Choose some zz such that z∈[x]∩[y]z \in [x] \cap [y]. Then, zRxzRx and zRyzRy, so xRyxRy. Thus for any tt, tRx⟹tRytRx \implies tRy due to transitivity, and tRy⟹tRxtRy \implies tRx for the same reason. So [x]=[y][x] = [y]. ◻

As an example, does there exist an equivalence relation on ℕ\mathbb N with three equivalence classes, two of which are infinite, and one of which is finite? Yes—we can break up ℕ\mathbb N into three parts, for example positive numbers, negative numbers and zero. This defines an equivalence relation.

Quotients

Given an equivalence relation RR on a set XX, the quotient of XX by RR is X/R={[x]:x∈X}X/R = \{ [x]: x \in X \} The map q:X→X/Rq\colon X\to X/R given by x↦[x]x \mapsto [x] is called the ‘quotient map’ or ‘projection map’. As an example, on ℤ×ℕ\mathbb Z \times \mathbb N, let us define (a,b)R(c,d)(a, b)R(c, d) to be true if ad=bcad=bc. This is an equivalence relation that demonstrates equivalence of rational numbers, where a,ca, c are the numerators and b,db, d are denominators. Here, ℤ×ℕ/R\mathbb Z \times \mathbb N / R is a copy of ℚ\mathbb Q, associating [(a,b)][(a, b)] with a/ba/b. Then, q:ℤ×ℕ→ℚq\colon \mathbb Z \times \mathbb N \to \mathbb Q would map (a,b)(a, b) to a/ba/b.
