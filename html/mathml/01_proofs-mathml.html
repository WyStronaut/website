Motivation for proof

Definition

A proof is a logical argument that establishes a conclusion.

Clearly there are some things missing from this definition; we have not yet defined a ‘logical argument’ or a ‘conclusion’; however we have to start somewhere, and assuming understanding of logic is a good place to start. There is a 3rd year course called ‘Logic and Set Theory’ that rigorously defines this.

There are two main reasons to want to prove things.

  1. To be sure that they are true; and

  2. to understand why they are true.

For the first point, it is easy to make a contrived example that shows why we need to prove statements even though they appear to be true for small nn, for example: ‘all positive integers nn are not equal to 100 trillion’. Understanding the reasoning behind why a statement is true is also very important; an example of this is at the end of this lecture.

Proofs and non-proofs

Claim

For any positive integer nn, n3−nn^3-n is a multiple of 3.

Proof

Given some positive integer nn, we have

n3−n=(n−1)n(n+1)n^3 - n = (n-1)n(n+1)

One of n−1,n,n+1n-1,\,n,\,n+1 must be a multiple of 3 as they are 3 consecutive integers.

Therefore, (n−1)n(n+1)(n-1)n(n+1) must be a multiple of 3. ◻

There are a couple of things to note about this proof.

  • The phrase ‘given a positive integer’ is important; we need to know where this variable nn came from.

  • We used the fact that three consecutive numbers contain a multiple of 3 here, but this was not proven. We must prove this fact elsewhere, or we cannot use it in this course!

  • It is important to write proofs legibly and linearly down the page; don’t just write a long line of symbols.

Claim

For any positive integer nn, if n2n^2 is even then nn is even.

Proof

Given a positive integer nn that is even, we have n=2kn=2k for some integer kk.

Thus n2=(2k)2=4k2=2(2k2)n^2 = (2k)^2 = 4k^2 = 2(2k^2),

so n2n^2 is even. ◻

Note

This is a false proof. We proved that B⟹AB \implies A, but we want A⟹BA \implies B. Our result wasn’t false, but it didn’t show what we set out to prove. The words ‘for some integer kk’ are important: we must specify which set kk belongs to. Our proof would be incorrect if we did not state this, as it would be unclear that 2(2k2)2(2k^2) is an even number.

Claim

For any positive integer nn, if n2n^2 is a multiple of 9 then nn is a multiple of 9.

Proof

Given a positive integer nn that is a multiple of 9, we have n=9kn=9k for some integer kk.

Therefore, n2=(9k)2=81k2=9(9k2)n^2 = (9k)^2 = 81k^2 = 9(9k^2),

so n2n^2 is a multiple of 9. ◻

Note

Not only does this fall for the same trap as the previous proof, but the original claim is false (e.g. n=6n=6)! It’s entirely irrelevant that the claim is true for some positive integers, because even one counterexample disproves the claim.

Let’s return now to the previous incorrect example: ‘if n2n^2 even then nn even for all positive integers nn’.

Proof

Suppose that nn is odd.

We have n=2k+1n = 2k+1 for some integer kk.

Therefore, n2=(2k+1)2=4k2+4k+1=2(2k2+2k)+1n^2 = (2k+1)^2 = 4k^2 + 4k + 1 = 2(2k^2 + 2k) + 1

n2n^2 is odd

Therefore nn is even. ◻

  • We prove things to show why something is true. We can see why this claim was true here—it’s really a statement about the properties of odd numbers, not the properties of even numbers.

  • We started by saying that we need something tangible to work with: just stating that ‘n2n^2 is even’ is really hard to work with because square roots just get messy and don’t yield any result. So we had to choose a clever first step.

  • The symbol shows that we have a contradiction.

This was a kind of proof by contradiction. Essentially, A⟹BA \implies B is the same as saying ¬B⟹¬A\neg B \implies \neg A. This is because:

  • A⟹BA \implies B means that there is no case such that AA is false and BB is true.

  • ¬B⟹¬A\neg B \implies \neg A means that there is no case such that ¬B\neg B is false and ¬A\neg A is true. In other words, there is no case such that BB is true and AA is false. This is equivalent to the case with A⟹BA \implies B.

Claim

The solution to the real equation x2−5x+6=0x^2-5x+6=0 is x=2x=2 or x=3x=3.

Note

This is really two assertions:

  1. x=2∨x=3⟹x2−5x+6=0x=2 \lor x=3 \implies x^2 - 5x + 6 = 0, and

  2. x2−5x+6=0⟹x=2∨x=3x^2 - 5x + 6 = 0 \implies x=2 \lor x=3

We can denote this using a two-way implication symbol ⇔\iff: x=2∨x=3⇔x2−5x+6=0x=2 \lor x=3 \iff x^2 - 5x + 6 = 0

Proof

We prove case i by expressing the left hand side as a product of factors: (x−3)(x−2)=0(x-3)(x-2)=0. The other case may be proven using factorisation. ◻

We can do another kind of proof using ⇔\iff symbols a lot. However, we need to be absolutely sure that each step really is a bi-implication.

Proof

For any real xx: x2−5x+6=0⇔(x−2)(x−3)=0⇔x−2=0∨x−3=0⇔x=2∨x=3\begin{align*} x^2-5x+6=0 & \iff (x-2)(x-3) = 0 \\ & \iff x-2 = 0 \lor x-3 = 0 \\ & \iff x=2 \lor x = 3 \end{align*} ◻

Claim

Every positive real is at least 1.

Proof

Let xx be the smallest positive real. We want to prove x=1x=1, so we prove this by contradiction.

Case 1: if x<1x < 1 then x2<xx^2 < x

Case 2: if x>1x > 1 then x<x\sqrt{x} < x

Therefore x=1x=1 ◻

Note

The assertion that there exists a smallest positive real is not justified. This means that the proof is invalid in its entirety. It is important that every line in a proof must be justified.
